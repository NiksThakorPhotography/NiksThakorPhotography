Kaise use kare — Deploy & Maintain (Step-by-step)

  1. GitHub account banaye: github.com par jaaye aur free account create karein.
  2. Repository banaye: New -> Repository; name: NiksThakorPhotography; Public; Create repository.
  3. Files upload karein: Repository page par "Add file" → "Upload files" → saare site files drag & drop karke commit karein.
  4. GitHub Pages enable karein: Repository → Settings → Pages (left menu) → Branch: main (root) → Save. Aapki site publish ho jaayegi at https://<your-github-username>.github.io/NiksThakorPhotography/
  5. Naya album add karna (manual, asaan):
    • Repo me albums/ folder banaye (agar nahi hai).
    • Ek naya folder banaye example albums/rahul-neha.
    • Folder ke andar ek subfolder banaye images/ aur apni photos wahan upload karein named 01.jpg, 02.jpg, 03.jpg.... (Sequential naming recommended)
    • Agar music chahiye to same album folder me ek file bgm.mp3 upload karein.
    • Repo me albums/albums.json file ko edit karein aur naya object add karein: 
      [
  {
    "id":"rahul-neha",
    "title":"Rahul & Neha",
    "date":"12 Sep 2025",
    "thumb":"albums/rahul-neha/thumb.jpg",
    "path":"albums/rahul-neha"
  }
]
  6. Album URL kis tarah share karein: Example: https://<your-github-username>.github.io/NiksThakorPhotography/albums/rahul-neha/ — ye link client ko bhej dein.
  7. Tip: Thumbnail (thumb.jpg) upload karein jo album listing me dikhai dega. Agar nahi hai to first image use hoti hai.

Agar aap chahen to mai aapko step-by-step GitHub par upload karte hue guide kar sakta hoon — main har click batata chalunga.
